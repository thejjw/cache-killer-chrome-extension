Configure Domain List
Add domains to include or exclude from cache killing. Use wildcards (*) for subdomain matching.
Examples:
• *.wikipedia.org - All Wikipedia subdomains (en.wikipedia.org, fr.wikipedia.org, etc.)
• example.com - Exact domain match
• localhost - Local development (matches all ports: localhost:3000, localhost:8080, etc.)
• *.google.com - All Google subdomains but not google.com itself
• google.com - Exact match for google.com

Note: Only domain names are matched. Ports are ignored, so "localhost" will match localhost:3000, localhost:8080, etc.
